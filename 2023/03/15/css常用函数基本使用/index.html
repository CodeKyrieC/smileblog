css常用基本函数

var()

创建css变量

rgb()

定义颜色：参数为 (red,green,blue) 0-255/0%-100%之间的值

rgba()

定义颜色并设置透明度：参数为 (red,green,blue,alpha) alpha值为0-1

attr()

访问元素属性的值，标签中 添加 data-自定名=”值” ，可接收三个参数：属性名、类型和默认值。

语法：attr(attribute-name?[,]?)

calc()

对css属性值进行计算

支持加减乘除，+ 和 - 俩边必须有空白字符

用于计算元素宽高，达到自适应效果

max()

需要两个值并使用该 CSS 属性的最大值。这对于制作响应式宽度和高度很有用。

blur()

毛玻璃(高斯模糊)效果，可应用于图片或元素

  • filter: 将模糊或颜色偏移等图形效果应用于元素

  • backdrop-filter:为元素后面的区域添加模糊或者其他效果

gradient()

是一种CSS数据类型的子类型，用于表现两种或多种颜色的过渡转变。

CSS的数据类型描述的是2D图形：比如background-image、list-style-image、border-image、content等， 常见的方式是通过url来引入一个图片资源；它也可以通过CSS的 函数来设置颜色的渐变。

常见的函数实现有下面几种：

1、linear-gradient()：创建一个表示两种或多种颜色线性渐变的图片

2、radial-gradient()：创建了一个图像，该图像是由从原点发出的两种或者多种颜色之间的逐步过渡组成

3、repeating-linear-gradient()：创建一个由重复线性渐变组成的<image>

4、repeating-radial-gradient()：创建一个重复的原点触发渐变组成的<image>
